## Causify.AI Projects

### Causify Helpers
- [GitHub](https://github.com/causify-ai/helpers)
- Core utilities and helper functions for Causify.AI's causal inference platform.
- Includes shared libraries for data processing, model utilities, and common
  operations used across the Causify ecosystem.

### Causify Tutorials
- [GitHub](https://github.com/causify-ai/tutorials)
- Educational resources and hands-on tutorials for learning causal AI concepts
  and using the Causify platform
- Includes Jupyter notebooks, example datasets, and step-by-step guides for
  predictive maintenance, demand forecasting, and other causal inference
  applications.

### Causify Platform (csfy)
- [GitHub](https://github.com/causify-ai/csfy)
- Main repository for the Causify.AI platform, implementing production-grade
  causal AI systems combining Bayesian inference, knowledge graphs, and temporal
  machine learning for decision-making under uncertainty.
